This is a simple electronic circuit simulator that I made in pure Java from scratch.

The simulator features only three types of components: wires, inverters, and power sources. Wires and power sources connect to any neighbor that they can, while inverters only connect to their neighbors which are directly in front and behind them.

I’ve implemented saving and loading of the grid, various keyboard shortcuts, and a helpful debug mode. I wanted to avoid using swing at all, for various reasons, and therefore implemented the buttons completely from scratch, rendering everything using the java.awt package.

This was a school project for Computer Science 20 and took around one month of in-school work.
