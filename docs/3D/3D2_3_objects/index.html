3 High Poly Objects


These are 3 objects I modeled and textured for a school assignment (at Digital Arts and Entertainment) as a way to learn subdivision surface modeling as well as procedural texturing. I spent around 4 weeks working this project.

Objects modeled:

  • Astro A40 Headset
  • Astro MixAmp Pro
  • Logitech G9X Laser Mouse

Tools used:

  • 3DS Max 2016
  • Photoshop CC 2015
  • Mental Ray Renderer
