Unreal Tournament 4 Level


This was a three month long personal project I began to challenge myself to a larger scale project and to force myself to learn how to use an efficient method of creating reusable props/materials.

I learned a lot about working quickly and using an iterative design process.

All prop modeling was done in Autodesk 3Ds Max 2017, and all texturing was done in Photoshop CC 2017.
